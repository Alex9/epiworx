Epiworx by mattruma

ASP.NET MVC, CSLA and C# based project, task and time management software

Dependencies

.NET, CSLA

Install

You'll need to run the Create Database script in the scripts folder, run it once for the epiworx database and then again creating an epiworx-test database (this is what the integration tests will hit).
